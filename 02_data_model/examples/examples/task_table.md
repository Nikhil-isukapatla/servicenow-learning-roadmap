# Task Table Example

## Overview

The Task table is the parent table for many applications in ServiceNow.

## Common Fields

| Field |
|--------|
| Number |
| State |
| Assigned To |
| Assignment Group |
| Priority |
| Short Description |
| Created By |
| Created On |

## Child Tables

- Incident
- Problem
- Change Request
- Catalog Task
- HR Case

## Why It Is Important

The Task table provides common fields and functionality that are inherited by child tables.

## Key Learning

Instead of creating the same fields repeatedly, ServiceNow uses table inheritance through the Task table.
