# Incident Table Example

## Overview

The Incident table stores records related to issues reported by users.

## Example Record

| Field | Value |
|--------|-------|
| Number | INC0010001 |
| Caller | John Doe |
| Priority | High |
| State | New |
| Short Description | Unable to access email |

## Explanation

- Each row in the table is called a **record**.
- Each column is called a **field**.
- Every incident has a unique Incident Number.
- The Incident table extends the **Task** table.

## Key Learning

The Incident table is one of the most important tables in ServiceNow and is widely used in ITSM.
